## Task Overview
This task focuses on setting up the Java environment, writing a basic Java program,
compiling it, and executing it using command-line arguments.

## Tools Used
- Java (OpenJDK 8+)
- IntelliJ IDEA Community Edition
- Git & GitHub
## Program Description
- Prints a welcome message
- Accepts command-line arguments
- Displays the arguments if provided

## How to Run the Program
1. Compile the program using javac
2. Run the program using java command
3. Pass command-line arguments during execution
## Output
- Without arguments: Displays default message
- With arguments: Displays each argument
